# ScrollGuard AI Agent Guide

This repository is designed to be implemented with AI coding agents.

Before writing code, the you MUST read the following documents in order:

1. `docs/PROJECT_SPEC.md`

   * Explains what the application is and its goals.

2. `docs/ARCHITECTURE.md`

   * Defines the system architecture and module responsibilities.

3. `docs/API_SPEC.md`

   * Defines the native Android API interface used by React Native.

4. `docs/TASKS.md`

   * Contains the step-by-step development tasks.

you should follow the tasks sequentially.

Rules:

* Implement tasks in order.
* Follow the architecture defined in `ARCHITECTURE.md`.
* When implementing native modules, follow the contracts in `API_SPEC.md`.
* Use TypeScript for React Native code.
* Use Kotlin for Android native modules.
* Do not change architecture unless necessary.

Development strategy:

1. Complete Phase 1 tasks first.
2. Commit after each task.
3. Ensure code compiles before moving to the next phase.

Goal:

Produce a working implementation of the ScrollGuard mobile application.
